## Инициализация данных (Seeding)
Перед запуском парсера необходимо наполнить справочники предметов и городов.
1. Убедитесь, что у вас есть файлы дампа `items.json` и `locations.json` (обычно берутся из репозитория `ao-bin-dumps`).
2. Запустите скрипт импорта (потребуется реализовать `src/scripts/seed_db.py`).
